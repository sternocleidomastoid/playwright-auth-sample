# Playwright Sample

A simple Playwright test suite to load and test a web application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. The `.env` file contains the base URL:
```
BASE_URL=http://localhost:5173/
```

## Running Tests

```bash
npm test
```

Or run with UI:
```bash
npm run test:ui
```

Or debug:
```bash
npm run test:debug
```
